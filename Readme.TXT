This project is developed in Eclipse IDE. To work on Eclipse IDE you just have to import this directory.
("File->Import->General->Existing project into workspace", then click next and choose this directory as the root.)

There are 2 external libraries in the project:
jsc: For probability and statistic functions.
opencsv-2.3: To read and write CSV files.

Graphs are placed into the "input" directory. To use one of them, simply rename the "filename" in SampleBasedCTP.java 
to select one of them.

Simulation results are saved into SimulationResults.csv.
